Installation

  1. Visit the Chrome Web Store.
  2. Click on the "Add to Chrome" button.
  3. Copy the Trial API key from Cohere here.
  4. Add the API key in the Extension Options of "Manage Extensions".

How to Use

Once the extension is installed, you can access it by clicking on the Rewrite icon in your Chrome toolbar. Copy the text that you want to rewrite and paste it into the context bin. Modify the options based on your requirements. For example, if you want to request an appointment via email, select "Email" in the first option and "Request" in the second. Customize your prompt as needed. Press the "Generate" button, and the extension will generate the email for you. Copy and paste it wherever you want!
